## Vending Machine

간단한 자판기 시뮬레이터입니다.
React와 MobX를 사용해 상태 기반 로직과 UI를 분리했고,
현금 결제, 카드 결제, 잔돈 계산, 환불, 관리자 기능을 포함합니다.

해당파일과 동일 경로에 자판기에서 사용자가 원하는 음료를 얻는 과정을 도식화한 "자판기 다이어그램.png"이 포함되어 있습니다.

---

## 실행 방법

1. 설치
   npm install

2. 개발 서버 실행 (Vite 개발 모드, 기본 포트: 5173)
   npm run dev
   브라우저에서 [http://localhost:5173](http://localhost:5173) 접속

## 기능 요약

결제 플로우

1. 결제 수단 선택 (현금 또는 카드)
2. 현금 투입 (단위별 버튼)
3. 상품 구매 (가능한 상품만 활성화)
4. 잔돈 계산

   * 가능: 상품 출고 + 잔돈 표시
   * 불가: 전액 환불 ("넣은 그대로 반환")
5. 관리자 기능

   * 음료 재고 보충
   * 현금 재고 보충
   * 최근 로그 및 결제 실패 사유 확인
